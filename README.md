# vuepress-starter

#### 介绍
磁吸动力是一个基于VuePress开发的开源文档社区平台,它包含Mac和Linux操作指令文档,同时也提供免费的各类操作平台的软件下载。
在如今AI喷井式爆发,我们未来会在vue强力驱动下结合ChatGpt人工智能模型,给用户带来更好的升级和体验。
